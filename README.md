# PauliStringsWitness

# List of the functions

## PauliStrings.m

Function to compute the Pauli strings for nc copies.

## maxwitSN2.m

Function to compute upper bounds on the Schmidt number witness using Pauli strings for nc = 2 copies.


## maxwitSN3.m

Function to compute upper bounds on the Schmidt number witness using Pauli strings for nc = 3 copies.
